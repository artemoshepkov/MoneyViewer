СПРАВКА ПО РАБОТЕ С MoneyViewer

1. Окно транзакций

    Добавить / удалить транзакцию.

      - Кнопка создания новой транзакции. Нажмите, выберите категорию и сумму траты для транзакции и нажмите [Ок].

      - Кнопка удаления транзакции.

    Фильтрация транзакций

      - Combobox для фильтрации по категории. Нажмите, в открытом списке выберите нужную категорию.

      - Combobox для фильтрации по дате. Нажмите, в открытом списке выберите за какой промежуток времени хотите увидеть транзакции.

      - Отображение даты и фильтрация транза по дате. С помощью кнопок ">", "<" переключайтесь на нужную вам дату.

    Баланс

      - Строка для отображения баланса.

2. Вкладка целей и аккаунтов

    Добавить, удалить цель / аккаунт

      - Кнопка добавления цели / аккаунта. Нажмите для добавления цели / аккаунта.

      - Окно добавления цели. Введите имя для цели, сумму, что нужно достичь и нажмите [Ок].

      - Окно добавления аккаунта. Введите имя для аккаунта и нажмите [Ок].

      - Кнопка удаления цели / аккаунта. Нажмите для удаление цели / аккаунта.

    Выбор аккаунта

      - Для выбора аккаунта два раза нажмите на нужный аккаунт, после чего название аккаунта покрасится в красный .

    Пополнение цели

      - Для пополнения цели два раза нажмите на нужную цель.

      - В появившемся окне введите сумму пополнения и нажмите [Ок].

3. График расходов

    График

      - График отображения расходов за год.

4. Вкладка целей и аккаунтов

    Диаграмма расходов

      - Диаграмма, отображающая расходы по категориям за выбранный период.

    Добавить, удалить категорию

      - Список категорий.

      - Кнопка добавления категории. Нажмите для добавления категории.

      - Окно добавления категории. Введите имя для категории и нажмите [Ок].

      - Кнопка удаления категории. Нажмите для удаление категории.
